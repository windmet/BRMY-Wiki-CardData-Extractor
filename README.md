# Wiki 卡牌数据全自动提取工具

这是一个用于提取、解析并格式化游戏卡牌数据的辅助工具，专为 Wiki 组编辑人员制作。

## 🛠️ 使用方法
1. 下载本项目发布的最新版 `AutoWikiBuilder.exe`。
2. 将游戏最新的 `master_data.s2b` 文件直接拖拽到 `AutoWikiBuilder.exe` 图标上。
3. 运行完成后，程序会在同目录下生成一份排版好的 `.xlsx` Excel 表格。
